# Sky Audio Market Intel - Staff Phone Install Guide

This app must be opened from an HTTPS website link to install on phones. It will not install properly from a `file://` link or directly from a ZIP file.

## iPhone / iPad

1. Open the HTTPS app link in Safari.
2. Tap the Share button.
3. Tap Add to Home Screen.
4. Tap Add.
5. Open Sky Intel from the home screen.

## Android

1. Open the HTTPS app link in Chrome.
2. Tap the three-dot menu.
3. Tap Install app or Add to Home screen.
4. Tap Install.
5. Open Sky Intel from the home screen.

## Important

Each phone stores its own app data in that browser. Use Export and Import to move data between devices. For one shared live team database, the app needs an online backend.
